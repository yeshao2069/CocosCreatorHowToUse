### Introduction
**Tween CC Utils** project created based on CocosCreator version 3.6.0 

### Support (Interface)
1. cc.fadeIn      -->  CCUtils.fadeIn
2. cc.fadeOut     -->  CCUtils.fadeOut
3. cc.show        -->  CCUtils.show
4. cc.hide        -->  CCUtils.hide
5. cc.toggleVisibility    -->  CCUtils.toggleVisibility

### Support (UI)
- Sprite 精灵图片
- Label 文本
- EditBox 输入框
- Toggle 复选按钮
- Particle2D 2D粒子
- Button 按钮
