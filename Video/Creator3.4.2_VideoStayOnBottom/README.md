### 说明
CocosCreator 3.0.0--3.3.2 版本，使用 stayOnBottom 功能是可以正常使用的，CocosCreator 3.4.x 版本，stayOnBottom 功能异常。

- 解决方案(临时)：在构建 web-mobile 后，手动修改 style.css，添加以下代码
```
position: absolute;
z-index: 1;
```
